This project showcases a comprehensive **Insurance Analytics Dashboard** built using **Excel, Power BI, and Tableau**.

## üß† Project Objective
To create an end-to-end insurance analytics solution that:

## üìà Key Performance Indicators (KPIs)
A total of **six KPIs** were developed and implemented consistently in all platforms.

| # | KPI | Description |
|---|-----|-------------|
| 1 | **Number of Invoices by Account Executive** | Tracks invoices generated by each executive |
| 2 | **Yearly Meeting Count** | Monitors annual client meeting frequency |
| 3 | **Cross-Sell (Target, Achievement, New)** | Evaluates performance of cross-sell opportunities |
| 4 | **New Business (Target, Achievement, New)** | Measures new client acquisition performance |
| 5 | **Renewal (Target, Achievement, New)** | Assesses renewal business performance |
| 6 | **Stage Funnel by Revenue** | Shows revenue progression across sales stages |
| 7 | **Top Open Opportunities** | Highlights top pending opportunities by revenue |

> *Note:* The KPI numbering in the presentation repeated ‚Äú3‚Äù; it‚Äôs consolidated here for clarity.

## ‚öôÔ∏è Calculations & Logic

| Metric | Formula | Source |
|---------|----------|--------|
| **Target** | Cross Sell + New + Renewal (from *Individual Budget*) | Excel / Power BI / Tableau |
| **Achievement** | Brokerage Amount + Fees Amount | Derived Measure |
| **Invoice** | Amount from *Invoice Table* | Raw Data |
| **% Achievement (Placed)** | `(Achievement / Target)` | Calculated |
| **% Achievement (Invoice)** | `(Invoice / Target)` | Calculated |
